## Overview

`luna` is an open-source, in-momery SQL interface to your object storage data. Think of it as an open-source alternative to Amazon Athena.

```sh
# Sample run:
$ RUST_LOG=info ./target/debug/luna
```

See [lunactl](https://github.com/flowerinthenight/lunactl/) for the cmdline.
